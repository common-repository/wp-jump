Jump to another WordPress site.

== Description ==

This utility plugin allow you to bookmark your site in admin bar.

This plugin is very useful if you have several, local or online, WordPress instalations.You can set up all sites admin bar, from one point.

So, you can jump fast from WP admin to WP admin.


**Features**

* Fast jump from WP to WP 
* Global register sites


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Jumlp'
3. Activate WP Jumlp from your Plugins page. 

= From WordPress.org =

1. Download WP Jumlp.
2. Upload the 'wp-css-generator' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WP Jumlp from your Plugins page.
